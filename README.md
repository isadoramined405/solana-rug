## 🔍 Understanding Safety Checks

The application reviews several factors before presenting a risk score. It looks at the technical setup of the token and its metadata.

### Mint Authority
The program checks if the creator can change the token supply. If the mint authority remains active, the creator can dilute your holdings at any time.

### Liquidity Locks
The software verifies if the liquidity pool holds locked funds. A locked pool prevents the creator from removing all funds after people buy the token.

### Ownership Renouncement
The analysis detects if the contract owner gave up control. An immutable contract provides more safety because no one can alter the rules of the token after it launches.

### Top Holders
It monitors the distribution of tokens. If a few addresses hold a massive share of the supply, those owners can dump their tokens and crash the price.
